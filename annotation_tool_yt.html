경계 판단 기준 — 3단계 질문

경계는 게임 세계의 상태가 의미 있게 전환되는 시점입니다. 아래 질문 중 하나라도 해당되면 경계가 아닙니다.

Q1
게임 상태가 변화하지 않았는가? 플레이어 조작이 아니라 게임 자체 상태가 달라져야 합니다.
게임 상태의 변화란 게임 시스템이 플레이어에게 무엇을 요구하는가가 바뀌는 것을 말합니다.
Q2
동일 상태 내의 미시 이벤트인가? 같은 상태 안에서 반복되는 작은 이벤트는 경계가 아닙니다.
예: 전투 중 개별 타격, 탐색 중 채집
Q3
게임 세계가 일시정지·대기 상태로 전환되지 않는가? 플레이어 조작이더라도 게임 세계가 멈추거나 대기 상태로 바뀌면 경계입니다.
예: 전투 중 인벤토리를 열어 게임이 일시정지되면 경계, 게임이 계속 진행되면 경계 아님.
세 질문을 모두 통과하면 경계로 표시하고, 이후 구간의 카테고리를 선택합니다.
카테고리 선택 기준

경계 이후 시작되는 구간의 성격을 선택합니다.

Exploration
자유 탐색, 이동, 퀘스트 수행 등
Combat
전투, 보스전 등 전투 상태
Narrative
컷씬, 대화, 스토리 연출
Menu/UI
메뉴, 로딩화면, 일시정지 등
Other
위 항목에 해당하지 않는 경우
판단 예시
✓ 경계 표시 (→ 카테고리)
  • 전투 시작 → Combat
  • 전투 종료, 탐색 복귀 → Exploration
  • 컷씬·대화 진입 → Narrative
  • 컷씬·대화 종료 → 이후 구간에 맞게 선택
  • 보스 페이즈 전환 → Combat
  • 새 지역·스테이지 진입 → Exploration
  • 로딩 화면 진입 / 종료 → Menu/UI
✗ 경계 아님
  • 전투 중 개별 타격·스킬 사용
  • 탐색 중 아이템 채집
  • 게임 세계가 계속 진행되는 중의 UI 조작
  • 카메라 이동 (게임 상태 변화 없음)
주의사항
이벤트 전환이 시작되는 시점을 표시합니다 (종료 시점이 아닌 진입 시점).
카테고리는 경계 이후 구간의 성격을 기준으로 선택합니다.
영상 전체에 걸쳐 동일한 기준을 적용하세요. 일관성이 가장 중요합니다.
애매한 경우엔 경계로 표시하고 Note에 메모를 남기세요.
